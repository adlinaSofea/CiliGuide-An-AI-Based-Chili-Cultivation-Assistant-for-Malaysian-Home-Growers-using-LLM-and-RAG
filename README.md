# CiliGuide-An-AI-Based-Chili-Cultivation-Assistant-for-Malaysian-Home-Growers-using-LLM-and-RAG
**Overview**

CiliGuide is an AI-powered web application developed to support Malaysian home gardeners and small-scale farmers in chili cultivation. The platform provides real-time cultivation guidance through a locally hosted Large Language Model (LLaMA) integrated with Retrieval-Augmented Generation (RAG) and ChromaDB for accurate, context-aware responses.

In addition, CiliGuide incorporates a Vision Language Model (VLM) for soil image analysis, enabling users to upload soil images and receive AI-assisted assessments and recommendations. The system aims to improve access to agricultural knowledge through an intelligent and user-friendly platform.

**Features**
1. Chat Advisory
2. Soil Analysis
3. Fertilizer and Care Guidance
4. Disease-Detection (text-based)
5. Post-Harvest Suggestion
6. Harvest Tracker
7. Community

The system consists of the following core components:

1. Web-based user interface for interaction.
2. FastAPI backend for request processing.
3. LLaMA-based advisory engine integrated with RAG.
4. ChromaDB vector database for knowledge retrieval.
5. Vision Language Model for soil image analysis.
6. Firebase for data management and storage.

**Project Objective**
1. To provide Malaysian home chili growers with an accessible and intelligent AI-powered cultivation assistant that delivers accurate and crop-specific guidance without requiring any hardware or technical expertise.
2. To empower users to manage and monitor their chili cultivation activities from planting through to harvesting through a structured and easy-to-use digital system.
3. To promote household food sustainability among Malaysians by making smart farming technology simple, affordable and available to everyone.
